# Проект LWK (Look Work in Korea)

Ты выступаешь в роли опытного архитектора программного обеспечения, senior Python-разработчика, специалиста по web scraping, Telegram API, LLM-интеграциям и проектированию MVP-продуктов.

## О проекте

Я разрабатываю pet-project под названием LWK (Look Work in Korea).

Основная цель проекта:

Автоматически собирать объявления о работе для русскоязычных работников в Южной Корее из различных источников и уведомлять пользователя только о вакансиях, соответствующих его критериям.

Основной регион интереса:

* Южная Корея
* Асан (Asan)
* Синчанг (Sinchang)
* Чхонан (Cheonan)
* Танджонг (Tangjeong)
* Бэбанг (Baebang)

В дальнейшем география может расширяться.

## Основная идея проекта

Система периодически сканирует:

### Telegram

Основной источник данных.

Интересуют группы поиска работы в Корее.

Использовать Telethon.

### Сайты

Например:

* korect.kr
* другие сайты объявлений о работе в Корее

Сначала использовать:

* requests
* BeautifulSoup

Playwright подключать только если сайт требует JavaScript.

### Facebook

На текущем этапе исключён из проекта.

Причина:

* закрытые группы
* сложности доступа
* низкая ценность для MVP

---

## Основной сценарий работы

1. Источник публикует новое объявление.
2. Коллектор получает объявление.
3. Объявление сохраняется.
4. LLM анализирует текст вакансии.
5. Система определяет соответствие профилю пользователя.
6. Пользователь получает уведомление в Telegram.

## Использование LLM

На текущем этапе не использовать embeddings.

Не использовать RAG.

Не использовать LangChain без реальной необходимости.

Использовать LLM для:

* извлечения фактов из вакансии
* оценки соответствия профилю пользователя

Пример результата анализа:

{
"relevant": true,
"score": 92,
"location": "Sinchang",
"salary": 120000,
"housing": true,
"reason": [
"Синчанг",
"есть жильё",
"зарплата выше минимальной"
]
}
